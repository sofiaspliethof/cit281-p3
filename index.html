Project 3

Project Information

Learning Objectives

  • Gain experience interpreting functional descriptions and specifications to complete an assignment
  • Gain experience breaking a project into manageable components
  • Gain experience writing and executing non-web server Node.js JavaScript code using VSCode
  • Practice creating and using code modules
  • Practice using modern JavaScript syntax
  • Gain experience writing and executing Node.js REST API server using VSCode
  • Gain experience using Express with the GET verb, routes, and route parameters
  • Gain experience working with static data
  • Gain experience testing code module without using a web server
  • Gain experience using Postman to test web server routes
  • Gain experience working with JSON

Summary

This project is about building a modular Node.js web application named the "Ultimate Coin Tool". This web application calculates U.S. coin combinations for a given amount and determines the total value of various coins. This project is an exercise to learn how to interpret functional specifications, structure code using modules, and use modern JavaScript features and JSON, as well as setting up a Node.js and Git repository, creating and testing two key functions (coinCombo and coinValue), and developing an Express web server with two GET routes.
